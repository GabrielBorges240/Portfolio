# Portfólio Pessoal

Portfólio web desenvolvido para apresentar meus projetos, habilidades técnicas e experiências na área de Tecnologia da Informação.

O site reúne aplicações desenvolvidas com **HTML, CSS, JavaScript e Python**, além de projetos que envolvem **integração com APIs, manipulação de dados e lógica de programação**.

## Acesse o projeto

🔗 https://gabrielborges240.github.io/WebPage-Portfolio/

## Objetivo

Centralizar e apresentar meus projetos acadêmicos e pessoais, demonstrando evolução em desenvolvimento web e programação.

- CSS3  
- JavaScript
